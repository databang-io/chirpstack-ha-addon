# Setup GitHub Repository for ChirpStack Add-on

## 1. Create GitHub Repository
1. Go to GitHub.com
2. Create new repository: `chirpstack-addon-repository`
3. Make it public
4. Don't initialize with README (we have one)

## 2. Add Remote and Push
```bash
cd /home/rodrigbe/DEV/perso/HA/chirpstack-addon
git remote add origin https://github.com/YOUR-USERNAME/chirpstack-addon-repository.git
git branch -M main
git push -u origin main
```

## 3. Add to Home Assistant
1. Go to Settings → Add-ons → Add-on Store
2. Click ⋮ (three dots) → Repositories  
3. Add: https://github.com/YOUR-USERNAME/chirpstack-addon-repository
4. Install ChirpStack add-on from the repository

## 4. Alternative: Manual Installation via File Editor
1. Go to Settings → Add-ons → Add-on Store
2. Install "File editor" add-on if not already installed
3. Use File editor to create: `/config/addons/local_chirpstack/`
4. Copy all files from /config/addons/chirpstack/ to /config/addons/local_chirpstack/
